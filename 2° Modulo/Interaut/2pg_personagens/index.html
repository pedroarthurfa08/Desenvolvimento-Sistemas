Precisa de ajuda?

Como selecionar uma habilidade:

  • Clique em um card de habilidade para selecioná-lo
  • Use as teclas de seta para navegar entre as opções
  • Pressione Enter para confirmar sua seleção
  • Use os botões de configuração para ajustar sua experiência

Atalhos de teclado:

  • Alt + A: Ativar/Desativar áudio
  • Alt + C: Alternar alto contraste
  • Alt + F: Aumentar/Diminuir fonte
  • Alt + H: Abrir/Fechar ajuda
  • Esc: Voltar
